Triggers

To find the objects described below, first hover over a product card and click the “Choose Shade” button. This will open a new modal in which you’ll find the following:

Select a Shade Button(s)

  • Hover On
  • Hover Off
  • On Click

Size Button(s)

  • Hover On
  • Hover Off
  • On Click

Add to Bag Button

  • Hover On
  • Hover Off
  • On Click

Rules

Select a Shade Button(s)

  • Hover On
    • Cursor changes to pointer.
    • Button slightly scales up to about 1.05 over 300ms with ease transition.
  • Hover Off
    • Cursor returns to default.
    • Button smoothly scales back to normal size.
  • On Click
    • A 2px white inner ring fades in quickly (about 200ms) to indicate selection.
    • Any previously selected shade ring fades out.
    • Only one shade can be active at a time.

Size Button(s)

  • Hover On
    • Cursor changes to pointer.
    • Button’s gray border transitions to black.
    • Button scales up slightly to about 1.02 over ~300ms (ease).
  • Hover Off
    • Cursor returns to default.
    • Border transitions back to gray.
    • Button smoothly scales back down to 1.00.
  • On Click
    • Button scales back down to 1.00 while keeping the black border.
    • Any previously selected size resets to default.

Add to Bag Button

  • Hover On
    • Cursor changes to pointer.
    • A pink colored overlay sweeps across the button from left to right over ~500ms with ease transition.
  • Hover Off
    • Cursor returns to default.
    • Pink overlay retracts leftward over ~500ms with ease transition.
  • On Click
    • Pink overlay expands to cover the button completely.
    • The button text animates up with staggered letters for ~500ms, transitioning to “Added.”
    • After roughly 5 seconds, the button resets to default state, with pink overlay retracting leftward and button text animating down with staggered letters to “Add to Bag.”
    • Product details (selected shade and size) are added to the database.
