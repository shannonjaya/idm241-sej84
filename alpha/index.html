Triggers

Add to Cart Button

To find the Add to Cart button, first open the Quick View modal for a product by clicking on the Add to Cart button in a product card. Then, you will find the Add to Cart button located on the bottom right of the Quick View modal.

Hover On

Hover Off

On Click

Rules

Hover On

The system checks whether a size has been selected to determine Add to Cart button state:

Inactive state (before size is selected)

  • Cursor changes to pointer.
  • Button background remains black with white text and does not animate.
  • After 2s delay, tooltip shows “Select Color Name, Width Category and Size.”.

Active state (after size is selected)

  • Cursor changes to pointer.
  • Button background fades from black to white with black 1px border.
  • Text fades from white to dark gray.
  • After 2s delay, tooltip shows “Add to Cart.”

Hover Off

Inactive state (before size is selected)

  • Cursor reverts to default.
  • Tooltip disappears.
  • Button background remains black with white text and does not animate.

Active state (before size is selected)

  • Cursor reverts to default.
  • Tooltip disappears.
  • Button background fades from white back to black, and text fades from dark gray to white.

On Click

  • System checks whether a size has been selected (color and width are preselected by default).
  • If no size is selected, the button action is blocked and red error text appears under the size selector.
  • If a size is selected, product data is added to the cart in the backend/database. The product modal closes, and the cart modal slides in from the right.
